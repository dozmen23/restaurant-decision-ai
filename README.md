# Restaurant Decision AI (Capstone Project)

This project aims to build an explainable restaurant decision system using:

- Google Maps data
- Review-based feature extraction
- MCDA (Multi-Criteria Decision Analysis)

## Current Features

- Baseline review analyzer
- Property extraction from user reviews:
  - food quality
  - staff friendliness
  - ambience
  - speed
  - cleanliness
  - view quality
  - date suitability

## Next Steps

- NLP-based extraction
- Confidence scoring
- MCDA integration
- Explainability layer
